Zero Trust Filters

Request-time servlet filters that enforce access control and challenge handling after authentication.

Access Control Filter

ZeroTrustAccessControlFilter evaluates authenticated requests against the persisted runtime decision state and handles allow, block, escalate, and related response paths.

Challenge Filter

ZeroTrustChallengeFilter coordinates challenge initiation for decisions that require additional verification or MFA continuation.

Repository and Runtime Context

The filter chain works together with AISessionSecurityContextRepository, MFA context integration, and action repositories that persist runtime zero trust outcomes.
