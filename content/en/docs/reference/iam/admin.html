Admin Console

Web-based administration console providing security monitoring, unified policy management (Policy Center), identity management (users/groups/roles/permissions), Zero Trust operations, and the visual Authorization Studio.

The Admin Console organizes features into logical groups. Some sections are conditionally visible based on edition (Enterprise, SaaS).

Policy Center

Access: /admin/policy-center

The Policy Center is the unified hub for all policy-related operations. It combines resources, policy creation, policy listing, and analysis tools into a single 4-tab interface.

Tab 1: Resources

Full resource workbench with inline editing, policy setup modal (Quick Grant / AI Builder), and AJAX actions (define, exclude, restore). Server-side pagination for large-scale deployments.

Tab 2: Policy Creation (3 Modes)

  • Quick Mode: Role-permission mapping with search and chip-based selection. Creates a basic ALLOW policy as a prerequisite for more complex policies. Server-side search API handles thousands of roles/permissions.
  • Manual Mode: Full policy form with dynamic Target (URL/METHOD), Rule, and Condition (SpEL) editing. Directly creates Policy/PolicyTarget/PolicyRule/PolicyCondition entities.
  • AI Mode: Natural language policy generation using SSE streaming. AI analyzes all existing roles, permissions, conditions, and policies to suggest optimal policy configuration. Results are validated and previewed before saving.

Tab 3: Policy List

Server-side paginated policy list with keyword search. Edit and delete actions with POST form submission.

Tab 4: Studio

Links to Authorization Studio, Policy Wizard, Resource Workbench (Enterprise), and summary statistics.

Access: /admin/security-monitor

Real-time security event viewer based on the audit_log table. All queries use DB-level pagination to handle millions of records.

Dashboard

The Dashboard (/admin/dashboard) provides a comprehensive security overview with data aggregated from audit_log over the last 24 hours.

Sections

  • Policy Status: Total policies, active resources, resource coverage
  • Zero Trust Decisions: Donut chart with ALLOW/CHALLENGE/BLOCK/ESCALATE/DENY breakdown
  • Authentication Stats: Semi-circle gauge showing success/failure ratio
  • Security Indicators: 2-column grid with drill-down links -- after-hours access, AI analysis, admin overrides, distinct IPs, average risk score, AI security errors
  • Threat Events: Recent high-risk events with direct links to Security Monitor detail
  • Blocked Users: Recently blocked users by the Zero Trust engine

Authorization Studio

Access: /admin/studio

A 3-panel visual tool for authorization analysis:

  • Explorer Panel: Search and browse users, groups, roles
  • Canvas Panel: AI natural language queries with SSE streaming + Cytoscape graph visualization (network/hierarchy/flowchart layouts)
  • Inspector Panel: Detailed analysis results, effective permissions, recommendations

Identity Management

Standard CRUD operations for Users (/admin/users), Groups (/admin/groups), Roles (/admin/roles), and Permissions (/admin/permissions).

The access chain: User -> Group -> Role -> Permission. Users inherit permissions through group membership and role assignment.

Role Hierarchies

Access: /admin/role-hierarchies

Parent roles automatically inherit all permissions from child roles. The hierarchy is resolved by Spring Security's RoleHierarchy and considered during both Authorization Studio analysis and CustomDynamicAuthorizationManager runtime evaluation.

Zero Trust Operations

Security Monitor

Access: /admin/security-monitor

Real-time audit log viewer with time-range filters (1h/6h/24h/3d), category filters (AI Analysis, Auth Failure, Admin Override), and DB-level pagination for millions of records.

Blocked Users

Access: /admin/blacklist

Manages users blocked by the Zero Trust engine. Supports filtering by status (BLOCKED, UNBLOCK_REQUESTED, RESOLVED, TIMEOUT_RESPONDED) and resolution workflows by administrators.

Workflow Wizards

Policy Wizard

Access: /admin/policy-wizard

3-step guided workflow: Select Role -> Choose Permissions -> Review and Confirm. Creates role-permission mappings with auto-generated ALLOW policies.

Granting Wizard

Access: /admin/granting-wizard

Membership management with real-time impact simulation. Shows permission gains/losses before applying changes.

AI Policy Generation

The AI policy generation system uses SSE streaming to create policies from natural language requirements.

API Endpoints

EndpointMethodDescription
/api/ai/policies/generate/streamPOSTSSE streaming policy generation from natural language
/api/ai/policies/generatePOSTSynchronous policy generation
/api/policies/build-from-business-rulePOSTSave generated policy to database

How It Works

  1. User enters a natural language requirement (e.g., "Allow developers to access /api/** during business hours")
  2. AdvancedPolicyGenerationLab enriches the request with all available roles, permissions, and conditions from the database
  3. PolicyGenerationContextRetriever searches vector DB for similar policy patterns (RAG)
  4. AI generates a PolicyResponse with roleIds, permissionIds, conditions, and ID-to-name mappings
  5. Frontend validates the response against available items and renders a preview
  6. User confirms and saves -- BusinessPolicyService creates Policy + PolicyTarget + PolicyRule + PolicyCondition entities
  7. CustomDynamicAuthorizationManager reloads and the policy takes effect immediately

Getting Started

A complete walkthrough from deployment to fully dynamic authorization:

StepActionDetails
1Deploy your applicationResource Scanner automatically discovers all endpoints
2Open Policy Center (Resources tab)Review discovered resources with inline editing
3Define permissionsClick "Define & Setup Policy" for resources that need authorization
4Create policiesUse Quick Mode (role-permission mapping), Manual Mode (Target/Rule/Condition), or AI Mode (natural language)
5VerifyPolicy takes effect immediately via hot-reload. Test with different user roles.
6MonitorUse Dashboard and Security Monitor for real-time security visibility
