보안 구성

Zero Trust, HCAD 탐지, SecurityPlane 에이전트 설정을 포함한 Contexa 보안 엔진의 구성 속성입니다.

Zero Trust 속성

security.zerotrust 아래의 속성으로, SecurityZeroTrustProperties에 바인딩됩니다. HCAD 임계값, 위협 평가 가중치, 적용 모드를 포함한 핵심 Zero Trust 엔진을 제어합니다.

전체 속성 참조는 메인 구성 페이지를 확인하세요.

관련 문서: Zero Trust 참조

HCAD 속성

hcad 접두사 아래의 속성으로, HcadProperties에 바인딩됩니다. 계층적 컨텍스트 인식 탐지(HCAD) 필터 파이프라인과 기준선 학습을 구성합니다.

관련 문서: HCAD 참조

속성 타입 기본값 설명
hcad
.enabled boolean false HCAD 필터 파이프라인 활성화 또는 비활성화
.filter-order int 0 서블릿 필터 체인에서의 HCAD 필터 순서
hcad.baseline.learning
.enabled boolean true 실시간 트래픽으로부터의 기준선 학습 활성화
.alpha double 0.1 온라인 기준선 적응 학습률

구성 예제

YAML 
hcad:
  enabled: true
  filter-order: 10
  baseline:
    learning:
      enabled: true
      alpha: 0.1

Security Plane 속성

security.plane 아래의 속성으로, SecurityPlaneProperties에 바인딩됩니다. AI 기반 보안 분석을 위한 SecurityPlane 에이전트와 LLM 실행 스레드 풀을 구성합니다.

속성 타입 기본값 설명
security.plane.agent
.name String SecurityPlaneAgent-1 이 에이전트 인스턴스의 고유 이름
.auto-start boolean true 애플리케이션 시작 시 에이전트 자동 시작
.organization-id String default-org 멀티테넌트 배포를 위한 조직 ID
.execution-mode String ASYNC ASYNC (비차단) 또는 SYNC (순차)
security.plane.llm-executor
.core-pool-size int 4 LLM 분석 작업을 위한 코어 스레드 풀 크기
.max-pool-size int 4 LLM 분석 작업을 위한 최대 스레드 풀 크기
.queue-capacity int 100 대기 중인 LLM 분석 작업의 큐 용량
YAML 
security:
  plane:
    agent:
      name: SecurityPlaneAgent-1
      auto-start: true
      organization-id: default-org
      execution-mode: ASYNC
    llm-executor:
      core-pool-size: 4
      max-pool-size: 4
      queue-capacity: 100

관련 문서: Zero Trust 참조, SOAR 참조

세션 보안 속성

security.session 아래의 속성으로, SecuritySessionProperties에 바인딩됩니다. 세션 고정 보호, 동시 세션 제한, 세션 타임아웃 동작을 포함한 세션 보안 설정을 제어합니다.

전체 속성 참조는 메인 구성 페이지를 확인하세요.

관련 문서: 상태 관리 참조
