IAM 구성

관리 콘솔 설정을 포함한 Contexa IAM (Identity and Access Management) 모듈의 구성 속성입니다.

IAM 구성 개요

IAM 모듈은 데이터베이스에 저장된 동적 정책을 통해 인가를 관리합니다. 대부분의 다른 Contexa 모듈과 달리, 핵심 정책 엔진은 application.yml을 통해 구성되지 않습니다 - 대신 정책은 Admin Dashboard UI를 통해 생성, 업데이트, 관리됩니다.

이 페이지에 문서화된 Properties 클래스는 관리 설정만 제어합니다: 관리 콘솔 REST 문서 경로. 정책 구성에 대해서는 Admin Dashboard 및 정책 관리 참조 페이지를 확인하세요.

IAM Admin 속성

contexa.iam.admin 아래의 속성으로, IamAdminProperties에 바인딩됩니다. 관리 모듈이 제공하는 REST API 문서 경로를 포함한 IAM 관리 콘솔 설정을 구성합니다.

속성 타입 기본값 설명
contexa.iam.admin
.rest-docs-path String /docs/index.html IAM 관리 모듈이 제공하는 REST API 문서 페이지 경로
YAML 
contexa:
  iam:
    admin:
      rest-docs-path: /docs/index.html
