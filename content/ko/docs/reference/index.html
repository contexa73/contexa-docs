홈 / 레퍼런스

API 레퍼런스

Contexa 플랫폼의 모든 모듈에 대한 완전한 API 문서입니다. 각 섹션에서는 Contexa를 Spring Boot 애플리케이션에 통합하는 데 필요한 공개 인터페이스, 클래스, 설정 프로퍼티 및 코드 예제를 다룹니다.

Core

AI 랩 오케스트레이션, LLM 통합, 파이프라인 실행, 모델 관리, RAG 운영, 스트리밍, 어드바이저 시스템 및 전략 패턴을 제공하는 기반 모듈입니다.

AI Lab

AI 워크로드를 위한 범용 랩 추상화입니다. AILab 인터페이스와 AbstractAILab 기본 클래스를 통한 동기, 비동기 및 스트리밍 처리를 제공합니다.

LLM Orchestrator

UnifiedLLMOrchestrator를 통한 계층형 모델 선택, 도구 호출 및 자동 어드바이저 통합이 가능한 통합 LLM 클라이언트입니다.

AI Pipeline

PipelineOrchestrator를 통해 컨텍스트 검색부터 LLM 실행, 응답 파싱까지의 설정 가능한 다단계 파이프라인입니다.

Model Providers

DynamicModelRegistry를 통해 우선순위 기반 선택을 지원하는 Ollama, Anthropic, OpenAI 동적 모델 레지스트리입니다.

RAG & Vectors

PgVector 통합, 캐시 레이어 및 도메인 특화 벡터 서비스를 갖춘 벡터 저장소와 유사도 검색입니다.

Streaming

StandardStreamingService를 통한 문장 버퍼링, JSON 추출 및 청크 처리 기능을 갖춘 Server-Sent Events 스트리밍입니다.

Advisor System

도메인 기반 구성, 활성화/비활성화 및 보안 컨텍스트 주입이 가능한 Spring AI Advisor 레지스트리입니다.

AI Strategy

분산 실행, 품질 게이트 및 폴백 메커니즘을 갖춘 DiagnosisType 기반 전략 라우팅입니다.

Security

실시간 위협 평가, 행동 이상 탐지(HCAD), 자율 차단, Shadow Mode 마이그레이션 및 포괄적인 이벤트 시스템을 제공하는 AI 기반 Zero Trust 보안 레이어입니다.

Zero Trust

AI 기반 요청 평가와 자율 보안 결정을 수행하는 핵심 Zero Trust 엔진입니다.

위협 평가

실시간 위협 점수 산정, 리스크 평가 및 다중 요소 위협 분석입니다.

HCAD

행동 패턴 분석을 위한 계층적 컨텍스트 이상 탐지(Hierarchical Contextual Anomaly Detection)입니다.

이벤트 시스템

보안 이벤트 발행, 구독 및 감사 추적 인프라입니다.

차단

설정 가능한 임계값과 에스컬레이션 정책을 갖춘 자율 차단 액션입니다.

Shadow Mode

모니터링 전용에서 완전 자율 적용까지의 4단계 마이그레이션 경로입니다.

Identity

인증 플로우를 위한 플루언트 DSL, 적응형 다중 인증(MFA), ASEP 어노테이션, Zero Trust Filter Chain 및 세션 상태 관리 기능을 갖춘 ID 관리입니다.

Identity DSL

인증 및 인가 플로우를 정의하기 위한 플루언트 빌더 API입니다.

인증

인증 프로바이더, 토큰 관리 및 세션 처리입니다.

적응형 MFA

TOTP, Passkey 및 OTT를 지원하는 리스크 기반 다중 인증입니다.

ASEP 어노테이션

선언적 보안 정책을 위한 AI Security Enforcement Point 어노테이션입니다.

상태 관리

세션 상태 추적, Redis 기반 영속성 및 상태 전이입니다.

Zero Trust 필터

Zero Trust 요청 인터셉션을 위한 서블릿 Filter Chain 통합입니다.

IAM

XACML 기반 정책 평가, @Protectable 어노테이션, 리소스 스캐닝, 권한 평가, 정책 작성 및 관리자 대시보드를 갖춘 ID 및 접근 관리입니다.

@Protectable

XACML 정책 적용을 통한 메서드 수준 인가 어노테이션입니다.

XACML 엔진

PDP, PEP, PIP, PAP, PRP 컴포넌트를 갖춘 완전한 XACML 3.0 정책 엔진입니다.

권한 평가기

역할 계층 구조 및 속성 기반 접근 제어를 지원하는 플러거블 권한 평가입니다.

정책 관리

정책 CRUD, 버전 관리, 충돌 해결 및 AI 지원 정책 작성입니다.

리소스 스캐너

시작 시 보호 가능한 리소스의 자동 검색 및 등록입니다.

관리자 대시보드

사용자, 역할, 그룹 및 정책을 위한 웹 기반 관리 인터페이스입니다.
