Zero Trust 필터

인증 이후 요청 시점에서 접근 제어와 챌린지 처리를 수행하는 서블릿 필터 체인입니다.

접근 제어 필터

ZeroTrustAccessControlFilter는 인증된 요청을 현재 런타임 결정 상태와 대조하고 allow, block, escalate 등 응답 경로를 처리합니다.

챌린지 필터

ZeroTrustChallengeFilter는 추가 검증이나 MFA 연속 처리가 필요한 결정에 대해 challenge 흐름을 시작합니다.

저장소와 런타임 컨텍스트

이 필터 체인은 AISessionSecurityContextRepository, MFA 컨텍스트 통합, 런타임 zero trust 결과를 저장하는 action repository와 함께 동작합니다.
